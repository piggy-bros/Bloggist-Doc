# Bloggist

> 简陋的博客 Web 应用
>
> ——by @PespiePie @yaody7

## 前端安装

```shell
# 安装依赖包
npm install

# 热加载的本地开发模式运行
npm run dev
```

## 后端安装

```shell
# 安装 Github 包
go get github.com/piggy-bros/bloggist-backend

# cd 进入目录后运行
cd bloggist-backend/
go run main.go
```
